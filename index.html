¿Que son las
Paginas web dinamicas
?

Definicion

Una página dinámica es una página que puede cambiar su contenido, manteniendo el mismo diseño y estilo.

Emplean diferentes lenguajes de programación.

La estructura de estas aplicaciones es más compleja.

Se administran desde un CMS (content management system).

Estos paneles de administrador posibilitan que cualquiera pueda actualizar las aplicaciones de forma sencilla, sin tener que depender constantemente de un especialista.

No requieren entrar a un servidor para modificarlas.

Los cambios o actualizaciones se pueden llevar a cabo en tiempo real desde el panel de administrador.
